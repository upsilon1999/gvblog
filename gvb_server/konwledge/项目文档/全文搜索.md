# 原理

构造如下形式的数据

```json
{
    "?id=标题1": {
        "body": "标题下的正文内容",
        "slug": "标题1的跳转路径",
        "title": "标题1"
    },
    "?id=标题2": {
        "body": "标题下的正文内容",
        "slug": "标题2的跳转路径",
        "title": "标题2"
    }
}
```

标准形式

```sh

```

较特殊的形式

